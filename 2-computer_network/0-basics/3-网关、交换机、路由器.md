# 网关、交换机、路由器

交换机 = 同一局域网内通信

路由器 = 不同网络之间通信

交换机工作在数据链路层，基于 MAC 地址转发，负责同一局域网内高速通信；

路由器工作在网络层，基于 IP 地址转发，负责不同网络之间的互联与路径选择。

只用交换机无法跨网络、上不了互联网；

只用路由器成本高、效率低、无法组建大规模局域网。

因此必须两者配合，构成现代网络的基础架构。

网关不是具体设备，是一个 “逻辑角色”

一个网络访问另一个网络的出口

通常就是路由器的 LAN 口 IP，网关通常由路由器担任，所以家里的设备网关地址就是路由器 IP。

设备要出局域网，必须把包发给网关，由网关转发

网关是一个逻辑概念，指网络出口，通常由路由器充当。

设备要访问外网，必须将数据包发送给网关（路由器），再由路由器转发到外部网络。
