# DNS 域名系统

将域名解析为 IP 地址

缓存

- 浏览器
- 操作系统
- 路由器

浏览器缓存 → 系统缓存 → 路由器缓存 → LDNS → 根 → 顶级域 → 权威 → 返回 IP

四层结构

- 根
- 顶级域
- 权威域
- 本地

DNS 同时使用两种协议：

UDP 53 端口：绝大多数 DNS 查询（快、无连接、足够可靠）
TCP 53 端口：
主从域名服务器之间区域传送（Zone Transfer）
响应报文超过 512 字节时
DNSSEC 相关操作
一句话背诵：DNS 默认用 UDP 53，大数据 / 同步用 TCP 53。

DNS 完整流程（递归 + 迭代）

浏览器查浏览器缓存
查系统 hosts 文件
查本地 DNS 服务器（LDNS）
本地 DNS 查根域名服务器（Root）
根返回顶级域服务器（.com）
顶级域返回权威域名服务器（Authoritative）
权威服务器返回IP
LDNS 缓存并返回给用户

查询方式（2 种）

递归查询（Recursive）
客户端 → LDNS
客户端只发一次请求，LDNS 全权负责查到结果再返回

迭代查询（Iterative）
LDNS → 根 → 顶级域 → 权威
每次服务器只返回下一跳地址，不替你继续查

递归是我交给你，你帮我搞定；迭代是你告诉我下一个问谁，我自己去问。
