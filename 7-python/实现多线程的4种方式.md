# Python 实现多线程的 4 种方式

1. threading.Thread

直接创建线程，传入目标函数。

```python
import threading
import time

def task(name):
    print(f"线程 {name} 开始")
    time.sleep(1)
    print(f"线程 {name} 结束")

if __name__ == "__main__":
    t1 = threading.Thread(target=task, args=("A",))
    t2 = threading.Thread(target=task, args=("B",))

    t1.start()
    t2.start()

    t1.join()
    t2.join()
```

2. 继承 threading.Thread 类

重写 run() 方法，适合逻辑复杂的线程。

```python
import threading
import time

class MyThread(threading.Thread):
    def __init__(self, name):
        super().__init__()
        self.name = name

    def run(self):
        print(f"线程 {self.name} 开始")
        time.sleep(1)
        print(f"线程 {self.name} 结束")

if __name__ == "__main__":
    t1 = MyThread("A")
    t2 = MyThread("B")
    t1.start()
    t2.start()
    t1.join()
    t2.join()
```

3. concurrent.futures.ThreadPoolExecutor（线程池）

适合批量任务，自动管理线程，代码更简洁。

```python
from concurrent.futures import ThreadPoolExecutor
import time

def task(name):
    print(f"线程 {name} 开始")
    time.sleep(1)
    print(f"线程 {name} 结束")
    return f"结果 {name}"

if __name__ == "__main__":
    with ThreadPoolExecutor(max_workers=3) as executor:
        # 提交多个任务
        future1 = executor.submit(task, "A")
        future2 = executor.submit(task, "B")
        
        # 获取返回值
        print(future1.result())
        print(future2.result())
```
