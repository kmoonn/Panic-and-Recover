1. 主键索引（聚簇索引）
叶子节点存：整行数据
一张表 只能有一个
列必须 NOT NULL + 唯一
默认就是主键
查询速度最快，不需要回表
关键字：唯一、非空、只有一个、存整行数据

2. 唯一索引
叶子节点存：主键
列的值 必须唯一，可以有一个 NULL
一张表可以有多个
等值查询极快
要回表
关键字：唯一、允许一个 NULL、可多个、要回表

3. 普通索引（二级索引）
叶子节点存：主键
最普通的索引
不要求唯一
加速查询、排序
必须回表
关键字：不唯一、加速查询、要回表

4. 联合索引（复合索引）
多个字段一起建索引
如：index(a,b,c)
遵循 最左前缀原则
可以是普通联合索引，也可以是唯一联合索引
能实现覆盖索引，避免回表
关键字：多字段、最左前缀、覆盖索引
