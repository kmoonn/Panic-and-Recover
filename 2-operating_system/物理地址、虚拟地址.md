# 物理地址、虚拟地址

物理地址（Physical Address）真正的硬件内存地址，CPU 寻址物理内存用的地址。

虚拟地址（Virtual Address）程序运行时看到的地址，每个进程都有独立的虚拟地址空间。编程时用到的指针、变量地址都是虚拟地址。

地址翻译
由 CPU 硬件 MMU（内存管理单元） 完成。
虚拟地址 → MMU → 物理地址。

虚拟地址空间：每个进程私有的、独立的地址范围。
物理地址空间：实际硬件内存条的地址范围。

程序使用虚拟地址，硬件使用物理地址；
每个进程有独立虚拟地址空间，通过 CPU 的 MMU 完成虚拟地址到物理地址的转换；
现代操作系统主流采用分页机制实现地址映射。

程序用的是虚拟地址，硬件用的是物理地址；
每个进程有独立虚拟地址空间，由 MMU 完成地址翻译，实现隔离、安全与内存扩展。
