# HashSet

## 数据结构

- 哈希表（HashMap），**元素作为 key 存储**，value 是一个**固定的空对象**
- 不允许重复元素，利用 HashMap 的 key 不可重复的特性保证元素的唯一性
- **无序**存储
- 只能存储一个`null`值

## 特点

- **非线程安全**

多线程环境下需手动保证同步（如使用`Collections.synchronizedSet()`包装）。

- **元素去重原理**

依赖两个方法：

`hashCode()`计算哈希值，确定元素在哈希表中的存储位置。

`equals()`比较两个元素的哈希值，若相同则判断为同一元素。

自定义对象去重必须重写 hashCode() 和 equals() 方法。
