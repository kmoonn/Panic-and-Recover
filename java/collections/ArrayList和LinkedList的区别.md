# ArrayList 和 LinkedList 的区别

ArrayList 和 LinkedList 都是 Java 中常用的 List 实现类，但底层数据结构、性能特性和适用场景有很大差别。

## ArrayList

- 底层结构是**动态数组**`Object[]`。
- 元素连续存储在数组中。
- 插入/删除**中间元素**慢，随机访问快。
- 支持快速随机访问。
- 数组占连续内存，存储元素外加一定冗余容量。

## LinkedList

- 底层结构是**双向链表**`Node pre, next`。
- 每个元素是一个节点，包含数据和前后节点引用。
- 插入/删除**头尾元素**快，随机访问慢。
- 不支持快速随机访问，需要从头或尾遍历到指定位置。
- 每个节点村数据+前后节点引用，开销大。
