# 线程调度算法

线程调度本质是：**操作系统决定哪个就绪线程获得 CPU**。

## 先来先服务 FCFS

- 按照到达顺序执行
- 不可抢占
- 容易造成长作业阻塞短作业

## 最短作业优先 SJF

- 执行时间最短的线程优先
- 理论上平均等待时间最优
- 实际难以准确预估执行时间

## 优先级调度

- 按优先级高低调度
- 可抢占或不可抢占
- 低优先级线程可能饥饿

## 时间片轮转

- 每个线程分配固定的时间片
- 用完即切换
- 公平、适合分时系统
- 时间片大小影响性能

## 多级反馈队列 MLFQ

- 多个就绪队列，不同优先级
- 新线程先进入高优先级队列
- 根据运行行为动态调整优先级
