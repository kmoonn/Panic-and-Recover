# 基本数据类型

- 整型
  - byte
    - 1 字节
    - 0
  - short 
    - 2 字节
    - 0
  - int 
    - 4 字节
    - 0
  - long 
    - 8 字节
    - 0L
- 浮点型
  - float 
    - 4 字节
    - 0.0f
  - double 
    - 8 字节
    - 0.0d
- 布尔型
  - boolean 
    - 1 位
    - false
- 字符型
  - char 
    - 4 字节
    - '\u0000' 空字符，数值为 0

基本数据类型作为类的成员变量或静态变量时，会被自动赋予默认值。
但作为局部变量时不会有默认值，必须显示初始化，否则无法通过编译。
