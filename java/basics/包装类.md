# 包装类

- 整型
  - Byte
  - Short
  - Integer
  - Long
- 浮点型
  - Float
  - Double
- 布尔型
  - Boolean
- 字符型
  - Character

>  基本数据类型与包装类型的区别

- 定义不同，包装类属于对象，使用`new`初始化，基本数据类型不是。
- 包装类型**默认值**为 `null`，基本数据类型默认值为 `0`、`\u0000`、`false`。
- 包装类型可以接收 `null`，POJO 类的属性必须使用包装类型，接收数据库的`NULL`值才不会出错。
- 包装类型**支持泛型**，基本数据类型不支持。
- 存储方式和位置不同，基本数据类型存储在**栈**中；包装类分为实例和引用，引用在**栈**上，实例在**堆**中。

> 装箱、拆箱

- 装箱：基本数据类型值 ——> 包装类对象（调用 `valueOf()` 方法）
- 拆箱：包装类对象 ——> 基本数据类型值
- JDK 5.0 开始支持自动装箱/拆箱

> 缓存机制

- Byte、Short、Integer、Long、Character
- 整数型包装类缓存：`-128 ~ 127`
- Character 缓存：`0 ~ 127`
- Boolean 缓存: `true、false`
- 缓存上限可以通过配置 JVM 更改
- 只有`valueOf()`方法构造对象时会用到缓存，`new` 方法等不会使用缓存

> 不可变性

包装类对象一经创建，其内容（**所封装的基本类型数据值**）不可改变（类似 String 类）。
