# Spring 与 Spring Boot 的区别

Spring：一个功能强大的企业级开发框架，**核心是 IoC / AOP**，配置方式灵活但偏复杂。

Spring Boot：基于 Spring 的**快速开发框架**，通过**自动配置 + 约定优于配置**，极大减少配置工作。

|        |         Spring         | Spring Boot  |
|:------:|:----------------------:|:------------:|
|   定位   |          基础框架          |    快速开发框架    |
|  配置方式  | XML / 注解 / Java Config |  注解 + 自动配置   |
|  启动方式  |    需要部署到容器（Tomcat）     |  内嵌容器，直接运行   |
|  依赖管理  |        需手动管理版本         | Starter 统一管理 |
|  开箱即用  |           否            |      是       |
|  上手成本  |           高            |      低       |
|  生产就绪  |         需自行整合          | Actuator 内置  |
