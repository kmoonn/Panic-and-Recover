# Java 中实现并发同步的方法

实现并发同步的关键是**锁定共享资源、控制执行顺序**。

核心方法共分为 4 类。

## 1. 关键字
- `synchronized`修饰方法、代码块，隐式锁，简单易用。
- `volatile`保证可见性、有序性，不保证原子性，适合单写多读。

## 并发工具类
- `ReentrantLock`显式锁，支持公平锁、可中断、超时锁。
- `ReadWriteLock`读写分离锁。

## 原子类

- `AtomicInteger`
- `AtomicReference`

基于 CAS 无锁机制，保证原子操作。

## 其他

- 控制线程执行顺序
- 限制并发数
- 使用线程安全容器
