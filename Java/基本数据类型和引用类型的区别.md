# 基本数据类型和引用类型的区别


| 特性       | 基本类型   | 引用类型                |
|----------|--------|---------------------|
| 存储位置     | 栈/寄存器  | 栈上存引用，堆上存对象         |
| 占用空间     | 固定     | 变量固定大小，对象大小可变       |
| 赋值方式     | 值复制    | 引用复制                |
| 比较方式     | == 比较值 | == 比较引用，equals 比较内容 |
| 是否可以调用方法 | 否      | 是                   |


## 基本数据类型

`byte, short, int, long, float, double, char, boolean`
- 存储在**栈**上或者**直接作为变量的值**（存储在寄存器）。
- 每个变量直接保存真实的**数据值**，不存在引用或者指针概念。
- 内存占用固定，每种类型都有固定大小。
- 赋值复制操作会复制值本身，不会影响原变量。
- 传参是值传递。

## 引用（对象）类型
`String、Integer、自定义类、接口、数组`等
- **栈**上**变量**存的是**对象的引用（内存地址）**
- 真正的对象内容存储在**堆**上。
- 内存占用不固定，引用大小固定，对象大小根据字段和内容而不同。
- 赋值操作复制的是引用，多个变量可以指向同一个对象。
- 传递的是引用，但引用本身是值传递。
