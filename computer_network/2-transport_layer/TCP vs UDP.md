# TCP vs UDP

- 是否面向连接
  - TCP 面向连接、三次握手四次挥手
  - UDP 无连接
- 可靠性
  - TCP 可靠传输，ACK（确认应答）、重传、流量控制、拥塞控制
  - UDP 不可靠，尽最大努力交付
- 是否有状态
  - TCP 有状态，维护连接信息
  - UDP 无状态，开销小
- 传输效率
  - TCP 开销大，效率低
  - UDP 开销小，速度快
- 传输形式
  - TCP 面向字节流
  - UDP 面向报文
- 首部大小
  - TCP 最少20字节
  - UDP 固定8字节
- 广播/ 多播
  - TCP 仅支持单播
  - UDP 支持单播、多播、广播
