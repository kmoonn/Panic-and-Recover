# WebSocket

基于TCP、应用层协议、全双工通信

支持服务器主动推送

一次握手，持久连接，双向实时数据传输

应用场景：视频弹幕、实时消息推送、实时游戏对战、多用户协同编辑、社交聊天

WebSocket 是一种在单个 TCP 连接上实现全双工、长连接的通信协议，允许服务器主动向客户端推送消息。

对比 HTTP：
HTTP： 单向、短连接、客户端问一次服务端答一次
WebSocket： 双向、长连接、两边随时可以发消息

WebSocket 解决的就是：实时推送、低延迟、高并发、节省服务器资源的双向通信问题。

WebSocket 是基于 TCP 的全双工长连接通信协议，
通过一次 HTTP 握手建立连接后，客户端与服务器可随时双向通信。
它解决了 HTTP 单向、无状态、无法主动推送的问题，
替代低效的轮询方式，实现低延迟、高并发、轻量的实时通信。
